# 📊 Call Center Analytics Dashboard

This project presents an interactive **Call Center Analytics Dashboard** developed using **Microsoft Power BI**.

The dashboard analyzes call center data to provide insights into **customer support performance, agent productivity, call trends, and service efficiency**.

The goal of this project is to transform raw call center data into meaningful visual insights that help organizations improve **customer service quality and operational performance**.

---

# 🛠 Tools & Technologies

- Microsoft Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Call Center Dataset (Kaggle)  
- Data Visualization & Business Intelligence  

---

# 📈 Dashboard Overview

The project contains **three analytical dashboard pages** designed to provide a complete overview of call center performance.

## 1️⃣ Call Center Overview

This page provides a **high-level summary of call center operations**.

Key metrics include:

- Total Calls Handled  
- Resolved vs Unresolved Calls  
- Average Call Duration  
- Call Volume Trends  
- Overall Performance Indicators  

---

## 2️⃣ Agent Performance Analysis

This section focuses on analyzing **agent productivity and efficiency**.

Key insights include:

- Agent-wise Call Handling Performance  
- Number of Calls Handled by Each Agent  
- Resolution Rate by Agent  
- Average Handling Time  

This helps identify **top-performing agents and areas that require improvement**.

---

## 3️⃣ Call Insights & Trends

This dashboard analyzes **customer issues and call patterns**.

Key insights include:

- Call Distribution by Issue Type  
- Call Resolution Status  
- Call Duration Analysis  
- Call Trends Over Time  

This helps in understanding **customer behavior and service performance**.

---

# 💡 Key Insights

- Identifies **call volume trends and peak hours**
- Evaluates **agent productivity and performance**
- Analyzes **resolved vs unresolved call patterns**
- Helps improve **customer support efficiency**
- Provides insights into **common customer issues**
